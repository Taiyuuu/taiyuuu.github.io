Overview

Eco-Budgeter is a modern financial tracking application designed to help users manage their personal finances while monitoring their environmental impact. By combining traditional expense tracking with "Eco-Scores," the app encourages sustainable consumer behavior.

Getting Started

Live Application

The application is deployed and ready to use at:

Core Features

AI Receipt Scanner

Harnesses Google Gemini AI to process images. Simply snap a photo to extract:

  • Automated store identification and categorizing.
  • Itemized price breakdowns and total calculation.
  • Eco-Score generation based on item sustainability.
  • Context-aware eco-tips for better future choices.

Smart Dashboard

Real-time financial monitoring with automated insights:

  • Proportional budget tracking (Spending vs. Ideal Daily Goal).
  • AI Coach: Personalized advice based on spending trends.
  • Interactive Recharts for categories and payment methods.
  • One-click CSV exports for advanced record keeping.

Social Networking

Gamify your journey toward sustainability with friends:

  • Global Eco-Leaderboard to compare average scores.
  • Nudges: Send instant messages to encourage your peers.
  • Friend discovery via username and request system.
  • Live presence and sync for competitive tracking.

Premium Customization

A fully adaptable interface tailored to your preferences:

  • Gradient Builder: Define colors, stops, and directions.
  • Light, Dark, and System theme synchronization.
  • Dynamic font scaling for enhanced accessibility.
  • Customizable profile bios and avatar uploads.

Technical Architecture

Frontend

  • Next.js (App Router): Core framework for routing and SSR.
  • Tailwind CSS: Utility-first styling for responsive design.
  • Lucide React: Icon library.
  • Custom Cursor: Interactive mouse follower that disables on touch devices.

Backend & Services

  • Supabase: Handles Authentication, Database (PostgreSQL), and Storage for receipt photos.
  • Google Gemini AI: Processes receipt text/images and generates financial advice.

Mobile Support

The app is fully responsive. Key mobile optimizations include:

  • A fixed bottom navigation bar for small screens.
  • Automatic custom cursor disabling on touch events.
  • Horizontal scrollable tab bars for friends and messaging.
