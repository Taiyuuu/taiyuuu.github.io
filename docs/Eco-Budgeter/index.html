Overview

Eco-Budgeter is a modern financial tracking application designed to help users manage their personal finances while monitoring their environmental impact. By combining traditional expense tracking with "Eco-Scores," the app encourages sustainable consumer behavior.

Core Features

AI Receipt Scanner

Harnesses Google Gemini AI to process images. Snap a photo to extract store identification, itemized breakdowns, and sustainability Eco-Scores.

Smart Dashboard

Real-time financial monitoring with automated AI coaching. Track proportional spending versus daily goals with interactive charts.

Social Networking

Gamify your sustainability journey. Compare scores on a global leaderboard, discover friends, and send motivational nudges.

Premium Customization

Adapt the interface to your preference. Includes a dynamic Gradient Builder, system theme sync, and accessibility controls.

Technical Architecture

Stack Overview

  • Next.js: Core framework for routing and server-side rendering.
  • Supabase: PostgreSQL database, Authentication, and Storage.
  • Google Gemini AI: Large Language Model for receipt processing.
  • Tailwind CSS: Utility-first styling for high-performance UI.
