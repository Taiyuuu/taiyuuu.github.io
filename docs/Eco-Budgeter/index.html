Overview

Eco-Budgeter is a modern financial tracking application designed to help users manage their personal finances while monitoring their environmental impact. By combining traditional expense tracking with "Eco-Scores," the app encourages sustainable consumer behavior.

Getting Started

Local Development

1. Ensure you have Node.js installed.

2. Create a .env.local file with your Supabase credentials:

NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key

3. Run the application using the provided batch file:

run.bat

The batch file will wait for the server to be ready before automatically opening your browser.

Core Features

📷 AI Receipt Scanner

Uses Google Gemini AI to analyze receipt images. It automatically extracts store names, items, prices, and calculates an Eco-Score based on the sustainability of your purchases.

📊 Financial Dashboard

Real-time charts powered by Recharts showing your spending vs. proportional budget, expense categories, and payment method distribution.

🏆 Eco-Leaderboard

Gamify your savings. Connect with friends to see who has the highest average Eco-Score. Send "Nudges" (messages) to encourage your peers.

🎨 Customization

Personalize your experience with Light/Dark modes or create a "Custom Gradient" theme. Adjustable font scaling ensures accessibility.

Technical Architecture

Frontend

  • Next.js (App Router): Core framework for routing and SSR.
  • Tailwind CSS: Utility-first styling for responsive design.
  • Lucide React: Icon library.
  • Custom Cursor: Interactive mouse follower that disables on touch devices.

Backend & Services

  • Supabase: Handles Authentication, Database (PostgreSQL), and Storage for receipt photos.
  • Google Gemini AI: Processes receipt text/images and generates financial advice.

Mobile Support

The app is fully responsive. Key mobile optimizations include:

  • A fixed bottom navigation bar for small screens.
  • Automatic custom cursor disabling on touch events.
  • Horizontal scrollable tab bars for friends and messaging.
