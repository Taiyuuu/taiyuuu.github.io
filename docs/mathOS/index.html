System Overview

mathOS provides a windowed multitasking environment inspired by macOS. It manages a suite of specialized tools within a unified workspace.

Dynamic Dock

Launch and manage applications. Features include icon jumping on launch, drag-to-reorder, and context menus for window management.

Mission Control

Click the desktop background to see all open windows at once. Organize your workflow across multiple virtual desktops.

Omni Search

Use the search bar in the Calculator title bar to instantly find and launch any tool in the mathOS library.

The Calculator Core

The central hub of mathOS. It uses MathJax for high-fidelity LaTeX rendering and Math.js for precise calculations.

Key Features

  • Scientific Mode: Toggle via the ⌘ button to access advanced trigonometry, calculus, and memory functions.
  • Ghost Brackets: Visually balances parentheses in real-time. Use → to auto-close brackets.
  • Memory: Use mc, m+, m-, and mr to manage stored constants across calculations.
  • Precision: Computes up to 14 decimal places of accuracy.

AI Assistant Gemini

A specialized interface for the Google Gemini API, integrated directly into the workspace.

Setup

To use the AI Assistant, paste your Gemini API Key in the toolbar. This key is stored securely in your browser's localStorage.

Advanced Interactions

  • Model Switching: Switch between Gemini 1.5 Flash for speed and Gemini 1.5 Pro for complex reasoning.
  • Context Awareness: The assistant remembers previous messages in the current session for multi-turn conversations.
  • Visual Feedback: Features a custom "Thinking" animation and gradual text reveal for a more natural interaction flow.

The Tool Suite

Every tool in mathOS is a standalone application that communicates with the core system.

Tool Description
Grapher 2D function plotting with interactive zooming.

Unit Converter Instant conversion across dozens of physical units.
Timezones Global map with real-time day/night cycles and city clocks.

Keyboard Shortcuts

  • Enter Evaluate current expression
  • Esc Clear display (AC)
  • Shift Toggle 2nd/inverse functions in Scientific mode
  • Backspace Delete last character / token
  • Arrow Right Auto-complete parentheses
