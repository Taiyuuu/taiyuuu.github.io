Floor Planner Guide

🛠 Tools & Creation

DrawSwitch modes with W (Wall), C (Curve), or D (Freehand).
ShapesChoose Rectangle, Oval, or Door from the dropdown. Snaps with S.
FillPaint Bucket: Fills enclosed areas based on both drawings and walls.
TextClick to place a label. Click an existing label with the Select tool to edit.
ColorThe color picker sets the active color for all tools and text.

🖱 Selection & Editing

  • Move: Use the Select tool to drag any wall, shape, curve, or text.
  • Edit Vertices: Select a wall or curve. Drag handles to reshape. Right-Click a segment to add a point; Right-Click a handle to delete it.
  • Windows: Select a wall, hover over a segment, and press E.
  • Marquee: In Select mode, drag on empty space to select freehand drawings. Use handles to Rotate (Hold S to snap) or Resize (Hold Shift to lock ratio).

💾 Project & View

  • Undo/Redo: Reverse or restore actions using the toolbar buttons or Ctrl shortcuts.
  • Save/Load: Use Save to export a .json project file. Use Load to resume a previous project.
  • Clear: The canvas icon completely resets the canvas and project data.
  • 3D View: Walk through your design. Use WASD to move and Mouse to look.
  • Download: Export your plan as a high-quality .png image.
